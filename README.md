## Overview

This project is a python-based database for basic information about pokemon data from generation I to VII. It allow the user see all pokemon lunched until VII generation, a list of all main pokemon games lunched in the series and the relative famous side games from pokemon franchise. The user can also research information about a specific pokemon by name or general pokedex number and see how many pokemons are exclusive in each generation.

This program have the following options:

    1 - Show all pokemons by their main pokedex number    
    2 - Search Pokemon
      - By Number
      - By Name    
    3 - List all types of pokemon and list all pokemon from the choosen type    
    4 - Search all pokemon from a specific generation    
    5 - List of all pokemon games    
    6 - List of all pokemons games by generation    
    7 - Clean Screen

### Input
- `pokemon.csv` - A document with all pokemon generall info (name, japanese name, stats, type1, type2, generation, etc)
